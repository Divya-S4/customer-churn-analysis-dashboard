# Customer Churn Analysis Dashboard

## Project Overview
This project analyzes customer churn behavior in a telecom company using MySQL and Power BI. The objective is to identify high-risk customers, understand churn drivers, and provide business insights to improve customer retention.

---

## Tools & Technologies Used
- MySQL
- Power BI
- DAX
- SQL
- Data Visualization

---

## Key Features
- Data cleaning and transformation using MySQL
- Churn KPI analysis
- Revenue loss analysis
- Customer segmentation
- Interactive Power BI dashboard
- Executive business insights

---

## KPI Metrics
- Total Customers
- Churned Customers
- Churn Rate %
- Revenue Lost
- Average Monthly Charges

---

## Business Insights
- Month-to-month contracts show the highest churn.
- Customers with high monthly charges are more likely to churn.
- Fiber optic users contribute the highest revenue.
- Customers with lower tenure have higher churn probability.
